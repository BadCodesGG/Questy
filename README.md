# Questy — Releases

Download the latest **Questy** installer — a local Escape from Tarkov **PvE** quest & map tracker.

### → [Download the latest release](https://github.com/BadCodesGG/Questy/releases/latest)

Grab `Questy_<version>_x64-setup.exe`, run it, and launch Questy. The app updates itself
automatically when a new version is published here.

*This repository hosts release downloads only. It is an unofficial, fan-made tool and is not
affiliated with Battlestate Games.*
